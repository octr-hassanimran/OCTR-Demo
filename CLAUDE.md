# Claude Code — OCTR Demo

## What this repo is
- **Product**: OCTR Energy Optimization Demo (investor-ready frontend)
- **Stack**: Next.js 14 (App Router) + TypeScript + Tailwind + shadcn/ui + ECharts/Recharts + R3F

## Commands (local)
- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run lint` — lint

## Conventions (important)
- Prefer **small, reviewable diffs** over big rewrites.
- Follow existing patterns in `src/` (components, layout, naming).
- Keep UI changes consistent with the dark forest/matcha theme.
- Avoid adding new dependencies unless clearly needed.

## How to work in this repo
- If unsure where code lives, ask me to locate the file(s) first.
- When changing UI, include a short “what changed / why” summary.
- Don’t touch secrets. Never write API keys into the repo.
